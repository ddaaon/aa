## 🔥 行业动态摘要

### 1. “质变点”已至：AI 开始像人一样“上班”

> 火山引擎 FORCE 大会（2026.06.23）

- **豆包大模型 2.1 Pro** 发布，首次在芯片设计 RTL 测试中**连续运行近 18 小时**，自主完成 9 轮迭代验证。
- 其 Agent 能力支持 **500 余个智能体**在 3D 虚拟城市中同步协作生成建筑。
- 核心判断：AI 已从“即时问答”进阶为“**像人一样连续工作数小时**”的数字工程师。

### 2. 多模态与世界模型：AI 开始理解物理世界

- 豆包 **Seedance 2.5** 实现 **30 秒原生视频直出**，被视为通往“世界模型”的关键一步。
- 已落地场景：
  - **具身智能**：为机器人提供虚拟预演环境，合成稀缺训练数据。
  - **工业制造**：支撑数字孪生与异常场景合成。
  - **智能驾驶**：生成极端天气、罕见事故等“Corner Case”，补全路测盲区。

### 3. ToB 成为共识：价格战与差异化并行

- **字节跳动** AI 战略重心从 C 端流量转向 **ToB 企业服务**，覆盖 30 余个行业解决方案。
- **智谱 GLM-5.2** 开源，在 Code Arena 榜单登顶全球可用模型第一，推动其成为港股首个万亿 AI 公司。
- 市场分化趋势：
  - **Anthropic**：专攻 Coding 赛道
  - **智谱**：锁定 ToB 编程
  - **Kimi**：聚焦 C 端白领办公 Agent
